## ðŸ“– Theory
A **list** in Python is an ordered and mutable collection of elements enclosed within square brackets `[]`.

### Key characteristics of Python lists:
- Can store **multiple data types** (int, float, string, boolean, etc.)
- Supports **indexing** and **negative indexing**
- Elements can be **modified** after creation
- Provides several built-in methods such as:
  - `append()`
  - `insert()`
  - `extend()`
  - `remove()`
  - `sort()`
- Supports built-in functions like:
  - `len()`
  - `sum()`
  - `max()`
  - `min()`

Lists are widely used for storing and manipulating grouped data efficiently in Python programs.
